## Key Takeaways

1. **Cookie Store is Key**: `BasicCookieStore` automatically manages cookies like Python's `Session`
2. **Session Init is Critical**: Must visit pages before API calls to get cookies
3. **Browser Headers Matter**: NSE checks User-Agent, Sec-Fetch-*, Referer, etc.
4. **Auto-Recovery Works**: 403 errors trigger session re-init and retry
5. **Rate Limiting Essential**: Prevents NSE from blocking due to too many requests
